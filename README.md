# AO ( Advanced Output )

AO is a simple and optimized library for making output look good

# Install

## Static

```
$ sudo make install_a
```

## Dynamic

```
$ sudo make install_d
```

# Build

## Static

```
$ make build_a
```

## Dynamic

```
$ make build_d
```

# Uninstall

```
$ sudo make uninstall
```
